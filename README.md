# Reproducible NGS Analysis of AGXX®-Induced Oxidative Stress in Staphylococcus aureus
This repository contains the code, workflows, and documentation developed for the Bioinformatics and Cloud Computing project, focused on the reproduction and validation of next-generation sequencing (NGS) data analysis from the study:'The AGXX® Antimicrobial Coating Causes a Thiol-Specific Oxidative Stress Response and Protein S-bacillithiolation in Staphylococcus aureus' by Loi et al 2018.
The project aims to replicate key aspects of the original transcriptomic analysis, emphasising data reproducibility, transparent workflows and best practices in computational genomics.
The genome assembly and annotation for Staphylococcus aureus USA300_TCH1516 (accession number CP000730) were obtained from NCBI GenBank. [https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_000017085.1/]

The SDRF (Sample and Data Relationship Format) file, a tab-delimited metadata file that maps samples to their data files, was downloaded from the ArrayExpress functional genomics data collection. [https://www.ebi.ac.uk/biostudies/arrayexpress/studies/E-MTAB-7074?query=E-MTAB-7074.sdrf.txt]
All analyses were conducted using open-source bioinformatics tools, with workflows designed to be reproducible and portable across local and cloud-based computing environments.
